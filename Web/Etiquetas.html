Etiquetas de Encabezados o Titulos:

  • h1: Encabezado Principal.
  • h2: Segundo nivel de encabezado.
  • h3: Tercer nivel de encabezado.
  • h4: Cuarto nivel de encabezado.
  • h5: Quinto nivel de encabezado.
  • h6: Sexto nivel de encabezado.

Etiquetas de Texto y Formato

  1. Énfasis:

    • i: Texto cursiva, se utiliza para denotar texto que tiene énfasis y es semánticamente importante en el contexto del contenido. Lunes
    • em: Texto enfatizado.- Lunes

  2. Formato en Negritas

    • b: Texto en negrita, ejemplo: Martes
    • strong: Texto fuerte, ejemplo: Martes

  3. Subrayado y tachados

    • u: Texto Subrayado, ejemplo: Miercoles
    • s: Se usa para marcar texto que ya no es relevante o es incorrecto, sin un contexto de eliminación o edición, ejemplo: Texto tachado

  4. Otras Etiquetas

    • mark: Texto resaltado, ejemplo: Texto resaltado
    • small: Texto en tamaño pequeño, ejemplo: texto pequeño
    • sub: Texto en subíndice. ejemplo: 210
    • sup: Texto en superíndice, ejemplo: 210

Etiquetas de Enlaces

  • a: enlace (hipervínculo), ejemplo: Enlace de Prueba

Etiquetas de Listas

  1. Listas Ordenadas

    • ol: Lista ordenada (numerada)
    • li: Elemnto de lista.

  2. Listas Desordenadas

    • ul: Lista desordenada (viñetas)
    • li: Elemento de lista.

Etiquetas de Tablas

  • table: Tabla
  • tr: Fila de tabla.
  • td: celda de tabla.
  • th: Encabezado o titulo de celda
  • Ejemplo:
    Nombre Edad Ciudad
